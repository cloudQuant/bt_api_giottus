# bt_api_giottus

Giottus exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_giottus
```

## Usage

```python
from bt_api_giottus import GiottusExchangeDataSpot, GiottusRequestDataSpot
```

## Features

- REST API support
- API key header authentication
- Symbol format: `BTCINR` or venue-native symbol strings
